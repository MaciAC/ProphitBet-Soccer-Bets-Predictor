Importance plots

Use "Importance Weights" method to train a classification model and plot its importance scores for each feature (column).
"Variance Analysis to plot variance scores for each feature." Features with very low variances have little predictive power, since the expected value of that feature is always similar.
Univariate test are statistical tests, which assign an importance score to each feature.
Finally, feature elimination process is a process on which a model is trained multiple times, each time removing a feature. The importance scores are computed by the model's accuracy at each iteration.
